Compile-time unrolled loop that applies a callable with a compile-time index sequence.

Compile-time unrolled loop that applies a callable with a compile-time index sequence.This function template performs a static (compile-time) iteration from 0 to N-1, invoking the provided callable f with the expanded indices as template arguments. It forwards any additional arguments to the callable.

Template Parameters
NNumber of iterations (compile-time constant).
FType of the callable object.
ArgsTypes of additional arguments forwarded to the callable.
Parameters
fCallable to invoke with template indices.
argsAdditional arguments to forward to the callable.
Returns
The result of the callable invocation (if any).
Note
The callable f must have a templated call operator accepting template parameter pack of std::size_t.
struct Printer
{
template <std::size_t... Is>
void operator()(int base) const
{
((std::cout << "Index " << Is << ": " << base + Is << '\n'), ...);
}
};
templa::static_for<3>(Printer{}, 10);
// Output:
// Index 0: 10
// Index 1: 11
// Index 2: 12
#pragma once
namespace templa
{
template <std::size_t N, typename F, typename... Args>
[[nodiscard]] constexpr auto static_for(F &&f, Args &&...args)
{
return []<std::size_t... I>(std::index_sequence<I...>, F &&f, Args &&...args) constexpr
{
std::forward<F>(f).template operator()<I...>(std::forward<Args>(args)...);
}(std::make_index_sequence<N>{}, std::forward<F>(f), std::forward<Args>(args)...);
};
};
