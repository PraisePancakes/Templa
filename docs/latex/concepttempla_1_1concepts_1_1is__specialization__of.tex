\doxysubsection{Detailed Description}
Checks whether a given type {\ttfamily T} is a specialization of the class template {\ttfamily Template}. 

This concept is satisfied if {\ttfamily T} is an instantiation of {\ttfamily Template} with any number of template arguments.\hypertarget{concepttempla_1_1concepts_1_1is__specialization__of_autotoc_md0}{}\doxysubsubsubsection{\texorpdfstring{Example}{Example}}\label{concepttempla_1_1concepts_1_1is__specialization__of_autotoc_md0}
