\doxysubsection{Detailed Description}
Concept to check if a container supports erasing of elements. 

This concept checks if the container either\+:
\begin{DoxyItemize}
\item is a basic\+\_\+string and Default\+Erasable for its elements, or
\item is Allocator\+Aware and supports Allocator\+Erasable, or
\item is not Allocator\+Aware and supports Default\+Erasable.
\end{DoxyItemize}
